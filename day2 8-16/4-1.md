## if

if文は以下の内のフォ`マットで成されています。<br/>
elseは省略可能です。式にはブ`ル型（true、false）を返す式をdし、<br/>
式のY果がtrueの龊悉ifブロック内の文をg行します。式のY果がfalseの龊悉elseブロック内の文をg行します。<br/>

    if　（式）　{
    　　文（}数可）
    }　else　{
    　　文（}数可）
    }

if文は以下の内のフォ`マットのようにelse ifを使用し、<br/>
}数の式をg行し条件にあった文のみをg行させることができます。<br/>
else ifはいくつでもdすることができます。elseは省略可能です。まずifの式をg行し、falseであれば次のelse ifの式をg行します。<br/>
式のY果がtrueであればそのelse ifブロック内の文をg行します。式のY果がfalseであれば、次のelse ifの式をg行します。<br/>
すべてのelse ifの式がfalseであれば、elseブロック内の文をg行します。 <br/>
if、else ifなど各ブロックのいずれかの文がg行されるとif文全体のI理はK了します。他のブロックのI理をg行することはありません。<br/>


    if　（式）　{
    　　文（}数可）
    }　else if　（式）　{
    　　文（}数可）
    }　else if　（式）　{
    　　文（}数可）
    }　else　{
    　　文（}数可）
    }

【例１】if文を使用した例です<br/>

      public static void main(String[] args) {
        int score = 70;   //(1)
        char grade;

        if (score >= 90) {   //(2)

          grade='B';
        } else if (score >= 50) {
          grade='C';

3.else ifの式で（score >= 70）を判定し、trueのためgrade = 'B'の文をg行後、if文I理をK了します。<br/>
4.printlnメソッドで涫gradeのを表示します。<br/>

式にはブ`ル型を返す式をdし、trueの龊悉衔1をfalseの龊悉衔2をg行します。<br/>
文1が返すデ`タ型と文2が返すデ`タ型には互Q性がなければなりません。<br/>

    式?:文１：文2；

【例２】?:演算子を使用した例です。

        int score = 80;   //(1)
        System.out.println("あなたは" +

      }
    }

【解h２】<br/>
1.涫scoreに80を代入します。<br/>
2.printlnメソッド内で？：演算子を使用します。score　>=　80はtrueのため"合格"を返します。<br/>

## switch文

switch文は以下の内のフォ`マットで成されています。if文との`いはif文が○○以上、○○と同等などブ`ル型のを判断基胜趣工毪韦し、<br/>

caseコマンドのには定数もしくは定数を返す式をdします。が一定でない涫をdすることはできません。<br/>

switch 文の幼鳏趣筏皮悉蓼switchにかれている（式）を判eし合致するcaseの文をg行します。<br/>
もしすべてのcaseに合致しない龊悉 defaultにdされている文のg行を行います。<br/>
breakはcase文の後に置き、そのcase文のg行後、switch文全体のI理からiけ出すために使用します。 <br/>

default、breakは省略可能です。breakがない龊稀case文をg行した後でもswitch文全体のI理からiけ出ず、次のcaseの判定を行います。<br/>
意淼膜搜}数のcase文をg行させる龊悉虺き、各case文の後にbreakをdします。<br/>

    switch　（式）　{
    　　case　：　文（}数可）；　break；
    　　case　：　文（}数可）；　break；
    　　case　：　文（}数可）；　break；
    　　default：　文（}数可）；　break；
    }
