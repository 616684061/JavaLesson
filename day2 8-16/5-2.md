# Rり返し文

## while文
主に指定した回数だけRり返し目的で利用されるfor文にして、条件が氦郡丹欷皮い腴gは何度でもRり返す目的で利用されるのがwhile文です。<br/>
式は次の通りです。<br/>

	while (条件式){
	  g行する文1;
	  g行する文2;
	  ...
	}
	
while文では条件式をuしtrueだった龊悉摔稀{」から「}」のブロック内に述された文をg行します。<br/>
条件式はvS演算子及び理演算子などを使って述します。そしてブロック内のI理が一通りKわると改めて条件式をuします。<br/>
そしてまたtrueだった龊悉摔膝芝恁氓内の文のg行し、falseだった龊悉摔while文をK了します。<br/>

Rり返しでg行される文が一つだった龊悉摔稀{」と「}」を省略して次のように述することもできます。<br/>

	while (条件式)
	  g行する文;
	  
このようにwhile文はfor文と同じように使用することができます。<br/>
for文とはなり初期化式や浠式がありませんので条件式で涫を使用する龊悉while文よりも前で述する必要があります。<br/>
条件式を涓するにはブロック内で自分で述する必要があります。<br/>

ではwhile文はどのような龊悉耸褂盲工毪韦と言えばRり返し回数がQまっておらず、<br/>
条件式でuするが不tに浃铯毪瑜Δ胜猡韦扦埂：gな例で言えばユ`ザ`からパスワ`ドなどを入力してもらい一致するま<br/>でRり返し入力を行うような龊悉浈匹スト文などをiみzみ、含まれていた文を1行ずつI理するような龊悉胜嗓扦埂<br/>
ただ、for文と同じような目的でwhile文を使用してもいません。<br/>
回数がQまっているrにはfor文で回数が未定の龊悉while文というのはあくまで目安です。<br/>

ではgな例を使いwhile文がどのようにI理されるのかを_Jしてみます。<br/>

	int i = 0;
	
	while (i < 2){
	  System.out.println("i = " + i);
	  i++;
	}
	
この龊悉洗韦韦瑜ΔI理の流れとなります。<br/>

	 1)i = 0;
	 2)条件式をu。iは2よりも小さいのでRり返しをg行
	 3)「i = 0」を画面に出力
	 4)i++;
	 5)条件式をu。iは2よりも小さいのでRり返しをg行
	 6)「i = 1」を画面に出力
	 7)i++;
	 8)条件式をu。iは2よりも小さくないのでRり返しをK了
	 
for文と同じくwhile文ではまず最初に条件式がuされます。もしも条件式がいきなりfalseとなった龊悉摔侠Rり返しI理はg行されません。<br/>
条件式がtrueだった龊悉摔while文の「{」から「}」までのブロック内の文をにg行していきます。<br/>

ブロック内の最後の文をg行したら1回目のRり返しがK了です。そしてwhile文の最初にり改めて条件式をuします。<br/>
trueなら再度ブロック内の文をg行し、falseだった龊悉摔while文の次の文へI理が移ります。<br/>

基本的にfor文と同じですが条件式を浠させるのはブロック内で述する必要があることを忘れないようにして下さい。<br/>
