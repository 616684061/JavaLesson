## インクリメント(++)、デクリメント(--)演算子の使用

++、--はそれぞれ象となるオペランドを1加ないしは1p少させます。演算子をオペランドの前に置く前置(++x、--x)と演算子<br/>

前置の龊悉が1加ないしは1p少した後にそのがuされます。後置の龊悉がuされた後に、<br/>
そのが1加ないしは1p少します。それぞれの振る舞いを以下に例示します。<br/>

**【例１】++、--演算子の前置の龊悉柔嶂盲龊悉握瘠胛瑜い芜`いを例示します。**

        public static void main(String[] args) {
            int x = 10;
            int y = 10;

            //涫xのを1加させた後、涫answerXに代入を行う。
            int answerX = ++x;   //①

            //涫yを涫answerYに代入した後、涫yのを1加させる。

## +演算子の文字BYとしての使用
**+**演算子は象となるオペランドのI方が数である龊悉霞铀悚趣筏普裎瑜い蓼工、<br/>
象となるオペランドの一方でもStringオブジェクトの龊悉稀+はStringオブジェクトをBYするものとして振舞います。<br/>
これまで何度も出てきたprintlnメソッドでこのC能は多用されています。System.out.println("答えは" + a);などです。<br/>
BYを行うオペランドのI方がStringオブジェクトであれば}はないですが、一方がStringオブジェクトで、<br/>
もう一方がStringオブジェクトでない龊悉悉饯违ペランドをStringオブジェクトにQする必要があります。<br/>
Qする方法はオペランドがオブジェクト型であるか基本型であるかによってなります。<br/>
オブジェクト型とはそれぞれのクラスから生成されたものでStringオブジェクトや各プログラマが作成する任意のオブジェクトなどがあります。<br/>
基本型とはchar、byte、short、int、long、float、double、booleanのデ`タ形式をもったものです。

オペランドがオブジェクト型の龊悉toStringメソッドを使用します。<br/>
toStringメソッドはデ`タをStringオブジェクトにQするメソッドであり、クラスのル`トであるjava.lang.Objectでg装されているため、<br/>
すべてのクラスで使用できます。オペランドが基本型の龊悉悉饯欷兢欷位本型に用意されているラッパクラス(int型がIntegerクラスなど)で<br/>
g装されているtoStringメソッドを使用して、基本型のデ`タをStringオブジェクトにQします。<br/>
