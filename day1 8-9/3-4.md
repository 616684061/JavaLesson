【例１】代入演算子を使用した例をdします。<br/>

    public class ExOperator12 {
      public static void main(String[] args) {
        int x1 = 10, y1 = 10;
        int x2 = 20, y2 = 20;
        int x3 = 30, y3 = 30;

        //(1)代入演算子の略形を使用
        x1 += 10;
        x2 &= 10;
        x3 <<= 10;

        //(2)略系を使用しない
        y1 = y1 + 10;
        y2 = y2 & 10;
        y3 = y3 << 10;

        //(3)それぞれの演算Y果を表示
        System.out.println(x1 + ":" + y1);
        System.out.println(x2 + ":" + y2);
        System.out.println(x3 + ":" + y3);
      }
    }

【解h１】<br/>
1.代入演算子の略形を使用し、演算I理を行います。<br/>
2.略形を使用せずに演算I理を行います。<br/>
3.略形を使用した龊悉仁褂盲筏胜った龊悉窝菟憬Y果を表示します。<br/>
